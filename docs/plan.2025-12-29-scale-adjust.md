# セル単位6ユニット化の全体スケール調整 計画

更新日: 2025-12-29

## 目的
- 1セル=6ユニットへ変更しつつ、既存の挙動と相対スケールを維持する
- 距離・速度・当たり判定などの線形スケールを0.5倍で統一する

## 方針
- セルサイズ変更に合わせ、線形距離/速度のみ0.5倍にスケーリングする
- 角度・時間・確率などの無次元値は変更しない
- 影響範囲はワールド生成、プレイヤー/NPC/ビット/ビーム、演出・SE距離まで含める

## ステップ
- [x] 距離/速度定数の洗い出し（world、main、flow、npcs、bits、beams、stage）
- [x] グリッド/ステージ/メインの距離関連定数を0.5倍に更新
- [x] NPC/ビット/ビーム/フローの距離・速度関連定数を0.5倍に更新
- [ ] 目視確認（移動体感、当たり判定、SE距離、部屋サイズ感）
