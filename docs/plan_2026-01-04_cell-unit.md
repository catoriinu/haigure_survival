# セル=1基準への全面移行 計画

更新日: 2026-01-04

## 目的
- 1セル=1の基準に変更し、実装上の「ユニット」概念を廃止する
- 既存のプレイヤー体感（距離感・速度感・当たり判定・演出）を維持する

## 方針
- 現在の1セル=6基準から、全ての線形距離/速度を1/6に縮小して一致させる
- 角度・時間・確率などの無次元値は変更しない
- 既存の`unitScale`定義は削除し、数値を直接置換する

- [x] グリッド基準値を1セル=1に変更（`cellSize=1`, `height=1`）し、ステージ生成の距離関連定数を1/6へ更新
- [x] プレイヤー/NPC/ビット/ビーム/演出の距離・速度・判定半径などを1/6へ更新し、`unitScale`を削除
- [x] 参照検索で`unitScale`や旧値の残りがないことを確認する
- [ ] 目視確認（移動体感、被弾判定、ビット形状、SE距離、部屋サイズ感、ミニマップ表示）
