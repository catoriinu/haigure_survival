# ビット発射エフェクト分解プラン

更新日: 2025-12-28

## 目的
- 円錐側面の走査発光 → 球への伝播 → 発射球の拡大 までを約0.3秒で表現する
- 発射方向は現行と同じ決定タイミングで固定し、命中率の変化を最小限に抑える
- 既存のビーム描画・当たり判定を大きく崩さずに演出だけを追加する
