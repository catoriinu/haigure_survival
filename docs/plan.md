# 光線命中時の光球サイズ調整 計画

更新日: 2026-01-25

## プロンプト
光線命中時の光球の大きさを決めるときに、そのキャラのキャラクタースプライトの大きさを考慮してほしいです。
キャラクタースプライトを完全に包み込むような光球にしてください。
おそらく、キャラクタースプライトの中心点を光球の中心にし、光球の半径は、キャラクタースプライトの中心点から角まで（+少しだけマージン）の長さにする必要があると思っていますが、よりよい計算方法があれば任せます。
計画を立ててから実装してください。

## ステップ
- [x] 影響箇所の把握（player/npcのヒットエフェクトと光球サイズ計算）
- [x] スプライトサイズから光球径を算出する処理を実装し、player/npcのエフェクト/ライト/判定に反映
- [x] 仕上げ（plan更新、差分確認）

## 結果
player/npcのヒット時光球サイズをスプライト幅/高さの対角線ベースで算出し、エフェクト球・ライト範囲・判定半径に反映。実行シーンのヒット演出も同じ計算で統一。
